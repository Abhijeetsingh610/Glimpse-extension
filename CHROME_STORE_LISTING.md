3. Use arrow keys to navigate
4. Press Enter to open

**Example Searches:**

- "GitHub repositories" - Find all GitHub-related items

- **AI Mode**: Natural language, context-aware (requires free Google Gemini API key)

## Frequently Asked Questions

**Q: Is Glimpse free?**
A: Yes! Glimpse is completely free and open source.

**Q: Do I need a Google API key?**
A: No! Local search works without any API key. The API key is optional for AI-powered search.

**Q: Is my data safe?**
A: Absolutely. Your data never leaves your device except when using AI mode (sent only to Google Gemini API).

**Q: Does this work offline?**
A: Yes! Local search mode works completely offline.

**Q: Can I customize the keyboard shortcut?**
A: Yes, go to `chrome://extensions/shortcuts/` to customize.

**Q: How many tabs/bookmarks can it search?**
A: Unlimited! It searches all your tabs and bookmarks.

**Q: Does this slow down my browser?**
A: No! Glimpse is lightweight and only runs when you use it.

**Q: Can I contribute to development?**
A: Yes! The code is open source on GitHub.
